# Focus Flow 알람 시스템 명세 (v1.7.6)

본 문서는 갤럭시 알람 스타일의 고도화된 사운드/진동 시스템 및 알람 동작 정책을 정의합니다. (v1.7.6: Z 플립 5 및 Android 14 최적화 반영)

## 1. 알람 트리거 및 제외 정책
- **최초 시작 시 제외**: 타이머 세션을 유저가 직접 처음 시작할 때(Start 버튼 클릭 시)는 알람이나 팝업을 띄우지 않습니다. 
- **넘기기(Skip) 동작**: 유저의 수동 행동이지만, 유저가 **'전체 화면 알람'**을 설정했거나 사운드가 **벨소리(Ringtone)**인 경우 설정을 존중하여 **전체 화면 알람**을 표시합니다. 그 외에는 상단 팝업으로 표시합니다.

## 2. 알람 모드 및 사운드 구성
- **알림 방법 (3종)**: 벨소리(Ringtone), 알림음(효과음), 진동(Vibration)
- **알람 노출 모드 (2종 유저 선택)**:
  1. **전체 화면 알람**: 기기 잠금/미사용 시에도 화면 전체 점유. (기본값)
  2. **상단 팝업 알람**: 화면 상단 헤드업 알림.

### [알람 노출 모드 결정 트리 (v1.7.6 최적화)]
1. **벨소리 우선 법칙**: 사운드가 **벨소리(Ringtone)**로 설정된 경우, 노출 모드 설정과 무관하게 **무조건 전체 화면 알람**으로 작동합니다. (수동 중지 UI 보장)
2. **전체 화면 설정 존중**: 설정에서 **'전체 화면 알람'**을 선택한 경우, 모든 상황에서 **전체 화면 알람**으로 작동합니다.
3. **팝업 설정 및 종료 알람**: **'상단 팝업 알람'** 선택 시, 벨소리가 아니라면 **종료 알람을 포함한 모든 상황에서 상단 팝업**으로 작동합니다.
4. **Android 14 & Z플립5 대응**: 기기가 접혀 있거나 잠겨 있는 상태에서도 전체 화면 알람이 최상단에 뜨도록 `showWhenLocked`, `turnScreenOn`, `WakeLock`을 복합적으로 사용합니다.

| 사운드/진동 설정 | 설정 탭 노출 모드 | 최종 노출 결과 | 비고 |
| :--- | :--- | :--- | :--- |
| **벨소리** | 전체 화면 OR 팝업 | **전체 화면** | 규칙 1 적용 |
| **알림음/진동만** | 전체 화면 | **전체 화면** | 규칙 2 적용 |
| **알림음/진동만** | 팝업 알람 | **상단 팝업** | 규칙 3 적용 |

## 3. 세션별 알람 패턴 및 문구
알람은 설정 탭에서 지정된 **집중, 휴식, 종료** 사운드가 각각 반영됩니다.

> **규칙**: 작업명이 없는 경우 **"독립 세션"** 명칭을 사용합니다. (예: `독립 세션: 몰입 시작`)

| 패턴 | 상황 | 알람 타이틀 (팝업/전체 공통) | 알람 내용 (전체 화면 전용) |
| :--- | :--- | :--- | :--- |
| **몰입 시작** | 휴식 종료 후 몰입 진입 | `{작업명}: 몰입 시작 (경과 시간)` | "흐름을 타고 집중을 시작할 시간입니다." |
| **휴식 시작** | 몰입 종료 후 휴식 진입 | `{작업명}: 휴식 시작 (경과 시간)` | "잠시 숨을 고르며 에너지를 충전하세요." |
| **몰입(연속)** | 이전 몰입 종료 후 다음 몰입 진입 | `{작업명}: 몰입 (경과 시간)` | "계속해서 집중을 유지해 보세요." |
| **종료** | 전체 세션 완료 | `{작업명}: 종료 (경과 시간)` | "모든 세션을 완료했습니다. 수고하셨습니다!" |

## 4. 사용자 경험(UX) 및 시스템 로직 (v1.7.6 패치)
- **즉각 중지 로직**: 알람창에서 **[중단]** 클릭 시 `NotificationHelper`의 `alertJob`을 즉시 취소하여 소리/진동 잔상이 남지 않도록 개선되었습니다.
- **화면 꺼짐 대응 (Android 14)**: 화면이 꺼진 상태(Doze 모드)에서 시스템 지연 없이 알람이 즉시 울리도록, 전체 화면 알람 시 0.5초 지연 로직을 제거하고 즉각적인 소리 재생을 보장합니다.
- **폴더블 최적화**: Z 플립 5의 커버 스크린 및 잠금 화면 가시성을 위해 `showForAllUsers`, `setInheritShowWhenLocked` 및 삼성 전용 메타데이터를 적용했습니다.
- **20초 타임아웃**: 유저 응답이 없으면 **20초** 후 사운드/진동이 자동 종료됩니다.
- **정보 표시**: 팝업 모드 시에는 간결함을 위해 내용을 생략하고 타이틀만 표시합니다.
