# 🌊 포커스 플로우 (Focus Flow)

> **"시간을 블록으로 그리다, 몰입을 흐름으로 만들다."**

**Focus Flow**는 막연하게 흘러가는 시간을 눈에 보이는 **'시각적 블록'**으로 설계하여, 당신의 집중력을 완벽한 흐름(Flow)으로 만들어주는 ADHD 특화 시각적 스케줄러입니다.

---

## 📖 초보자 가이드: 시간을 지배하는 블록 활용법

### 1. 📅 계획하기 (시간을 블록으로 그리다)
- **블록 선택 (앵커 시스템):** 시작 시간을 터치한 후 종료 시간을 터치하세요. 사이의 블록이 자동으로 연결됩니다.
- **범위 수정:** 중간 블록을 터치해 길이를 바로 조절하거나, 첫 블록을 다시 터치해 전체를 취소할 수 있습니다.
- **세션 생성:** 세션을 생성하면 타이머 탭으로 자동 이동하며, 데이터가 로드된 '대기' 상태가 됩니다.

### 2. ⏱️ 실행하기 (전략적인 몰입의 시작)
- **[NEW] 집중/휴식 사이클 (v1.4.0):** "50분 일하고 10분 쉬기"와 같은 커스텀 사이클을 지원합니다. 설정에서 휴식 시간을 지정하면 앱이 알아서 집중과 휴식 구간을 배치합니다.
- **세션 인터벌 고정:** 타이머 시작 시점의 설정을 박제하여, 도중에 설정을 바꿔도 현재 세션의 흐름이 깨지지 않도록 보호합니다.
- **넘기기(Skip):** 작업이 일찍 끝났거나 휴식을 일찍 마치고 싶다면 `넘기기`를 눌러 즉시 다음 구간으로 이동하세요.

### 3. 📋 관리하기 (일목요연한 스케줄)
- **시스템 레벨 정밀 알람:** `AlarmManager`를 통해 백그라운드에서도 정확한 시점에 구간 전환을 알려줍니다.
- **배터리 최적화 대응:** 설정에서 배터리 최적화 예외를 요청하여 앱이 중간에 종료되는 것을 방지하세요.

## ✨ 주요 기능 (v1.4.0)

- **[NEW] 집중/휴식 사이클 시스템**: 50/10, 25/5 등 고전적인 뽀모도로부터 커스텀 사이클까지 자유로운 구성 가능.
- **[NEW] 시각적 상태 구분**: 집중 중(파란색)과 휴식 중(보라색)을 컬러로 명확히 구분하여 인지 부하 감소.
- **세션 인터벌 고정**: 실행 중 설정 변경으로부터 현재 세션의 무결성 보호.
- **고정밀 타이머 시스템**: Doze 모드 대응 `AlarmManager` 기반 알람.
- **지능형 알림 시스템**: 구간 전환 및 세션 종료 시 진동/팝업 알림.
- **앵커 기반 블록 선택**: 직관적인 범위 지정 방식.

## ⚙️ 기본 설정 (Default Settings)

- **알림 단위(집중)**: 15분 (사용자 변경 가능)
- **휴식 시간**: 0~15분 (사이클 모드 활성화)
- **진동**: 활성화
- **최신 버전**: v1.4.0 (Code 29)
