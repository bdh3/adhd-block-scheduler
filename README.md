# 🌊 포커스 플로우 (Focus Flow)

> **"시간을 블록으로 그리다, 몰입을 흐름으로 만들다."**

**Focus Flow**는 막연하게 흘러가는 시간을 눈에 보이는 **'시각적 블록'**으로 설계하여, 당신의 집중력을 완벽한 흐름(Flow)으로 만들어주는 ADHD 특화 시각적 스케줄러입니다.

---

## 📖 초보자 가이드: 시간을 지배하는 블록 활용법

### 1. 📅 계획하기 (시간을 블록으로 그리다)
- **블록 선택 (앵커 시스템):** 시작 시간을 터치한 후 종료 시간을 터치하세요. 사이의 블록이 자동으로 연결됩니다.
- **범위 수정:** 중간 블록을 터치해 길이를 바로 조절하거나, 첫 블록을 다시 터치해 전체를 취소할 수 있습니다.
- **[HOT] 태스크별 독립 설정 (v1.4.2+):** 세션 생성 시 `[연속 집중]`과 `[인터벌 사이클]` 중 하나를 선택하세요. 각 태스크는 자신만의 알람 주기와 휴식 시간을 가집니다.

### 2. ⏱️ 실행하기 (전략적인 몰입의 시작)
- **전략적 사이클:** "50분 일하고 10분 쉬기"와 같은 커스텀 사이클을 지원합니다. 설정된 휴식 시간에 맞춰 앱이 알아서 집중과 휴식 구간을 배치합니다.
- **데이터 박제(Freeze):** 타이머 시작 버튼을 누르는 순간의 설정이 해당 세션에 고정됩니다. 도중에 설정을 바꿔도 현재 진행 중인 몰입의 흐름은 깨지지 않습니다.
- **넘기기(Skip):** 작업이 일찍 끝났거나 휴식을 일찍 마치고 싶다면 `넘기기`를 눌러 즉시 다음 구간으로 이동하세요.

### 3. 📋 관리하기 (일목요연한 스케줄)
- **시스템 레벨 정밀 알람:** `AlarmManager`를 통해 백그라운드에서도 정확한 시점에 구간 전환을 알려줍니다.
- **배터리 최적화 대응:** 설정에서 배터리 최적화 예외를 요청하여 앱이 중간에 종료되는 것을 방지하세요.

## ✨ 주요 기능 (v1.4.7)

- **[NEW] UI 사용성 및 정밀도 개선**: 인터벌 설정 슬라이더 5분 단위 스냅 적용.
- **[NEW] 알림 정보 명확화**: 알림창에 작업 이름과 현재 상태(집중/휴식)가 결합되어 표시.
- **[FIX] 알림 타이틀 정제**: UI용 시간 정보 대신 순수 작업명만 표시되도록 개선 및 "작업" 기본값 노출 버그 수정.
- **[FIX] 넘기기(Skip) 경과 시간 오류**: 구간 넘기기 시 누적 경과 시간이 0으로 표시되던 버그 수정.
- **[FIX] 타이머 탭 세션 로직 정밀화**: 일시정지 시 현재 작업만 노출, 중지 상태에서 작업 선택 전환 및 해제(토글) 정상화.
- **[FIX] 빌드 및 런타임 안정화**: 아이콘 리소스 모호성 해결 및 타이머 초기화 로직 보강.
- **[FIX] 독립 작업 격리**: 캘린더 설정이 독립 작업에 영향을 주지 않도록 리셋 로직 강화.

## ⚙️ 작업 모드 및 설정 안내

1. **단일 작업 기본 전략 (Settings)**
   - 계획 없이 즉시 시작하는 작업에 적용될 '기본값'을 설정합니다.
   - 여기서 설정한 값은 '추가' 버튼 클릭 시 기본 세팅으로 제안됩니다.

2. **연속 집중 모드 (Continuous Focus)**
   - 휴식 없이 설정된 주기마다 시간 경과만 인지하고 싶을 때 사용.

3. **인터벌 사이클 모드 (Interval Cycle)**
   - 집중과 휴식을 교차 배치하여 에너지 효율을 극대화 (예: 뽀모도로 25/5).
