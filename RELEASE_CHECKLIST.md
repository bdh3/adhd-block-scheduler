# 🚀 Focus Flow 출시 체크리스트 (Release Checklist)

출시 전, 아래 항목들을 반드시 순차적으로 확인하고 업데이트합니다.

## 1. 버전 및 정보 업데이트
- [ ] **build.gradle.kts**: `versionCode` 상향 및 `versionName` 최신화.
- [ ] **README.md**: 최신 버전 정보와 이번 릴리즈의 주요 기능/수정사항 반영.
- [ ] **UI 표시 확인**: 스플래시 화면, 설정 탭 등 사용자가 직접 보는 버전 정보가 최신인지 확인.

## 2. 코드 및 안정성 확인
- [ ] **디폴트 값 확인**: 이번 릴리즈에서 변경된 기본 설정값이 의도한 대로 (`SettingsRepository` 등) 세팅되었는지 확인.
- [ ] **권한 및 시스템 설정**: 배터리 최적화, 알림 채널, 필수 권한 요청 로직이 정상인지 확인.
- [ ] **영속성**: 앱 업데이트 시 사용자 데이터 및 설정값이 유실되지 않는지 로직 검토.

## 3. Git 및 배포
- [ ] **Git Push**: 모든 변경 사항을 원격 저장소에 푸시.
- [ ] **Git Tag**: `v1.5.0`과 같이 버전 네임을 태그로 생성하여 푸시. (`git tag v1.5.0`, `git push origin v1.5.0`)
- [ ] **Release APK**: GitHub 등 배포처에 태그와 일치하는 APK 업로드.
