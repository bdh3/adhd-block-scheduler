# Focus Flow 알람 시스템 명세 (v1.8.3)

본 문서는 갤럭시 알람 스타일의 고도화된 사운드/진동 시스템 및 알람 동작 정책을 정의합니다. (v1.8.3: 사운드 중복 해결 및 FSI 노출 안정성 강화 반영)

## 1. 알람 트리거 및 제외 정책
- **최초 시작 시 제외**: 타이머 세션을 유저가 직접 처음 시작할 때(Start 버튼 클릭 시)는 알람이나 팝업을 띄우지 않습니다. 
- **넘기기(Skip) 동작**: 유저의 수동 행동이지만, 유저가 **'전체 화면 알람'**을 설정했거나 사운드가 **벨소리(Ringtone)**인 경우 설정을 존중하여 **전체 화면 알람**을 표시합니다. 그 외에는 상단 팝업으로 표시합니다.

## 2. 알람 모드 및 사운드 구성
- **알림 방법 (3종)**: 벨소리(Ringtone), 알림음(효과음), 진동(Vibration)
- **알람 노출 모드 (2종 유저 선택)**:
  1. **전체 화면 알람**: 기기 잠금/미사용 시에도 화면 전체 점유. (기본값)
  2. **상단 팝업 알람**: 화면 상단 헤드업 알림.

### [알람 노출 모드 결정 트리 (v1.8.3 정밀화)]
1. **노출 설정 우선**: 설정에서 **'전체 화면 알람'**을 선택한 경우, 소리/진동 여부와 관계없이 **무조건 전체 화면**으로 작동합니다. (소리가 꺼져도 화면은 꽉 차게 표시)
2. **벨소리 예외 법칙**: **'상단 팝업 알람'** 설정 중이라도, 사운드가 **벨소리(Ringtone)**이고 **소리가 켜져(ON)** 있다면 **전체 화면**으로 강제 전환됩니다. (수동 중지 UI 보장)
3. **무음/진동 시 기본 준수**: 소리를 완전히 껐다면(OFF) 벨소리 예외가 적용되지 않으므로, 유저의 **노출 모드 설정(전체/팝업)**을 그대로 따릅니다.
4. **Android 14 & 폴더블 커버스크린 대응**: 잠금 상태에서도 `startActivity`와 `WakeLock`을 통해 화면 점유를 보장합니다.

| 설정 탭 노출 모드 | 소리/진동 설정 | 소리 ON/OFF | **최종 노출 결과** | 비고 |
| :--- | :--- | :--- | :--- | :--- |
| **전체 화면** | (무관) | (무관) | **전체 화면** | 규칙 1 적용 |
| **팝업 알람** | **벨소리** | **소리 ON** | **전체 화면** | 규칙 2 적용 |
| **팝업 알람** | **벨소리** | **소리 OFF** | **상단 팝업** | 규칙 3 적용 |
| **팝업 알람** | **알림음/진동** | (무관) | **상단 팝업** | 기본 원칙 |

## 3. 세션별 알람 패턴 및 문구
알람은 설정 탭에서 지정된 **집중, 휴식, 종료** 사운드가 각각 반영됩니다.

> **규칙**: 작업명이 없는 경우 **"집중 세션"** 명칭을 사용합니다. (예: `집중 세션: 집중 시작`)

| 패턴 | 상황 | 알람 타이틀 (팝업/전체 공통) | 알람 내용 (전체 화면 전용) |
| :--- | :--- | :--- | :--- |
| **집중 시작** | 휴식 종료 후 집중 진입 | `{작업명}: 집중 시작 (경과 시간)` | "흐름을 타고 집중을 시작할 시간입니다." |
| **휴식 시작** | 집중 종료 후 휴식 진입 | `{작업명}: 휴식 시작 (경과 시간)` | "잠시 숨을 고르며 에너지를 충전하세요." |
| **집중(연속)** | 이전 집중 종료 후 다음 집중 진입 | `{작업명}: 집중 (경과 시간)` | "계속해서 집중을 유지해 보세요." |
| **종료** | 전체 세션 완료 | `{작업명}: 종료 (경과 시간)` | "모든 세션을 완료했습니다. 수고하셨습니다!" |

## 4. 사용자 경험(UX) 및 시스템 로직 (v1.8.0 패치)
- **즉각 중지 로직**: 알람창에서 **[중단]** 클릭 시 `NotificationHelper`의 `alertJob`을 즉시 취소하여 소리/진동 잔상이 남지 않도록 개선되었습니다.
- **버튼 편의성**: `AlarmActivity`의 중단 버튼 높이를 **80dp**로 확대하여 긴박한 상황에서도 쉽게 끌 수 있도록 개선했습니다.
- **다크모드 배경**: 야간 눈부심 방지를 위해 배경색을 `0xFF0F172A`로 고정 적용했습니다.
- **종료 알람 안정성**: 시스템에 의해 종료 알람이 사라지지 않도록 `setOngoing(true)` 플래그를 유지하여 알람의 우선순위를 확보했습니다.
- **20초 타임아웃**: 유저 응답이 없으면 **20초** 후 사운드/진동이 자동 종료됩니다.
- **정보 표시**: 팝업 모드 시에는 간결함을 위해 내용을 생략하고 타이틀만 표시합니다.
