# 📝 Focus Flow 변경 이력 (Changelog)

모든 버전의 상세 변경 내역을 기록합니다. `README.md`는 최신 버전의 요약 정보만 제공합니다.

---

## [v1.8.4] - 2026-04-18
### Added
- **빌드 환경 안정화**: VS Code JDK 간섭을 차단하고 Android Studio 내장 JDK(JBR) 사용을 강제하여 빌드 신뢰성을 확보했습니다.

### Fixed
- **캘린더 네비게이션 동기화 (Patch 16)**: `HorizontalPager` 애니메이션 중 중간 날짜가 ViewModel을 오염시키는 현상을 `isScrollInProgress` 가드 및 `scrollToPage` 즉시 이동으로 해결.
- **캘린더 퀵 세션 생성 버그 수정 (Patch 17)**: 캘린더 퀵 추가 바에서 세션 생성 시 사용자의 휴식 시간 설정이 무시되던 문제를 수정했습니다.
- **날짜 계산 및 데이터 동기화**: 타임존 오프셋 방지 로직 강화 및 세션 생성 시 시간 표시 오류(19:00~19:00) 등을 해결했습니다.

## [v1.8.2] - 2026-04-10
### Added
- **설정 보호 UX**: 타이머 진행 중(`isTimerActive`) 주요 설정 항목 비활성화.

- **사운드 지연 튜닝**: 화면 상태별 벨소리 재생 지연 시간 정밀 조정(400ms/200ms).
