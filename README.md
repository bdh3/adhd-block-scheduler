# 포커스 플로우 (Focus Flow)

**Focus Flow**는 집중의 흐름을 유지하고 시간을 시각적으로 관리하는 안드로이드 앱입니다. 1시간 단위의 통합 타이머를 통해 사용자가 설정한 리듬에 맞춰 끊김 없는 몰입(Flow)을 유지하도록 돕습니다.

## 🌟 핵심 요약 (4-Line Description)

1.  **작업 중심의 1시간 몰입**: 선택한 할 일에 1시간 동안 깊이 몰입할 수 있도록 돕는 '끊김 없는' 사이클 타이머입니다.
2.  **유연한 시간 쪼개기**: 1시간을 1~60개 블록으로 자유롭게 쪼개어, 집중 시간 동안 규칙적인 알림으로 주의를 환기합니다.
3.  **자동화된 휴식 관리**: 집중과 휴식 비율을 설정하면 별도의 조작 없이도 다음 블록으로 자동 전환되어 흐름을 유지합니다.
4.  **시각적 성취 확인**: 전체 남은 시간 시각화와 캘린더 자동 기록을 통해 하루의 집중 성과를 직관적으로 관리합니다.

## ✨ 주요 기능

- **1시간 통합 타이머**: 블록마다 타이머가 멈추지 않고, 전체 세션이 끝날 때까지 하나의 흐름으로 이어집니다.
- **작업(Task) 기반 시작**: '진행할 작업'을 먼저 선택하여 목표를 명확히 한 뒤 몰입 세션을 시작합니다.
- **휴식 시간 기반 자동 설계**: 휴식 시간(0~30분)만 설정하면 남은 시간이 자동으로 집중 블록으로 구성됩니다.
- **블록 단위 알림**: 집중 시간 동안 설정한 쪼개기 단위(예: 1분/15분)마다 짧은 진동을 주어 잡생각을 방지합니다.
- **똑똑한 상태 전환**: 휴식 시간에는 중간 알림을 끄고 온전한 휴식을 보장하며, 휴식 시작/종료 시점에 명확한 알림을 제공합니다.
- **캘린더 자동 기록**: 완료된 집중 세션을 기기 캘린더에 자동으로 기록하여 시간 사용 내역을 시각화합니다.
- **백그라운드 안정성**: 배터리 최적화 제외 권한 연동으로 화면이 꺼진 상태에서도 타이머가 정확하게 작동합니다.

## ⚙️ 기본 설정 (Default Settings)

- **시간 쪼개기**: 4개 (15분 단위 알림)
- **휴식 시간**: 15분
- **작업 사이클**: 총 60분 중 **45분 집중(15분마다 3회 알림) + 15분 휴식**

## 🚀 빌드 및 실행 방법

1.  **Prerequisites**: Android Studio (최신 버전 권장), JDK 11 이상
2.  **Download APK**: [GitHub Releases](https://github.com/bdh3/adhd-block-scheduler/releases)에서 최신 버전(**v1.0.7**)의 APK를 다운로드할 수 있습니다.
3.  **SSH Clone**:
    ```bash
    git clone git@github.com-adhd-block-scheduler:bdh3/adhd-block-scheduler.git
    ```
