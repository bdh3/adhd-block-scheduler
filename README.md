# 🌊 포커스 플로우 (Focus Flow) (v1.7.6)

> **"시간을 블록으로 그리다, 몰입을 흐름으로 만들다."**

**Focus Flow**는 막연하게 흘러가는 시간을 눈에 보이는 **'시각적 블록'**으로 설계하여, 당신의 집중력을 완벽한 흐름(Flow)으로 만들어주는 혁신적인 시각적 스케줄러입니다.

---

## 📖 초보자 가이드: 시간을 지배하는 블록 활용법

### 1. 📅 계획하기 (시간을 블록으로 그리다)
- **블록 선택 (앵커 시스템):** 시작 시간을 터치한 후 종료 시간을 터치하세요. 사이의 블록이 자동으로 연결됩니다.
- **범위 수정:** 중간 블록을 터치해 길이를 바로 조절하거나, 첫 블록을 다시 터치해 전체를 취소할 수 있습니다.
- **태스크별 독립 설정:** 세션 생성 시 `[연속 집중]`과 `[인터벌 사이클]` 중 하나를 선택하세요. 각 태스크는 자신만의 알람 주기와 휴식 시간을 가집니다.

### 2. ⏱️ 실행하기 (전략적인 몰입의 시작)
- **전략적 사이클:** "50분 일하고 10분 쉬기"와 같은 커스텀 사이클을 지원합니다. 설정된 휴식 시간에 맞춰 앱이 알아서 집중과 휴식 구간을 배치합니다.
- **데이터 박제(Freeze):** 타이머 시작 버튼을 누르는 순간의 설정이 해당 세션에 고정됩니다. 도중에 설정을 바꿔도 현재 진행 중인 몰입의 흐름은 깨지지 않습니다.
- **넘기기(Skip):** 작업이 일찍 끝났거나 휴식을 일찍 마치고 싶다면 `넘기기`를 눌러 즉시 다음 구간으로 이동하세요.

### 3. 📋 관리하기 (일목요연한 스케줄)
- **시스템 레벨 정밀 알람:** `AlarmManager`를 통해 백그라운드에서도 정확한 시점에 구간 전환을 알려줍니다.
- **알림 채널 v3:** 무분별한 시스템 알람음을 차단하고, 앱 내에서 직접 몰입에 방해되지 않는 최적의 소리와 진동을 트리거합니다.

## ✨ 주요 기능 및 업데이트 (v1.7.6)

- **[NEW] 📱 갤럭시 Z 플립 5 & Android 14 완벽 호환**: 기기가 접혀 있거나 잠겨 있는 상태에서도 전체 화면 알람이 잠금 화면 위로 즉시 나타나도록 로직을 대폭 강화했습니다. (Z플립 전면/커버 스크린 가시성 확보)
- **[NEW] 🔄 네비게이션 UX 최적화**: 하단 탭 이동 시 뒤로가기 백스택이 무한히 쌓이는 현상을 제거했습니다. 또한, 타이머 미작동 시 앱에 재진입하면 세션 상태를 "작업을 선택하세요"로 자동 초기화하여 쾌적한 시작을 지원합니다.
- **[FIX] 🔊 알람 소리 잔상 완벽 제거**: 알람창에서 '중단'을 누른 후 약 0.5초간 소리가 더 나던 레이스 컨디션 버그를 해결했습니다.
- **[FIX] 🏷️ 작업 명칭 유실 및 저장 오류 수정**: 안내 문구("작업을 선택하세요")가 실제 작업명으로 저장되던 현상을 수정하고, 캘린더 세션 시작 시 작업명이 유실되던 버그를 잡았습니다.
- **[FIX] ✍️ 설정 화면 문구 정교화**: 알람 모드 설명에서 모호했던 조사와 불필요한 문구(알림 내용 생략 등)를 정리하여 가독성을 높였습니다.

## 🔔 알람 및 알림 상세 정책 (v1.7.6)

> **중요**: 알람에 관한 상세한 기술적 명세와 문구 정책은 [README_ALARM.md](./README_ALARM.md)를 최우선으로 참조하십시오. 본 섹션은 주요 요약 정보만 제공합니다.

### 🛠️ 동작 규칙 (Final Guidelines)
1. **알람 중단 정책**:
   - 전체화면 알람의 '중단' 버튼 클릭 시 즉시 종료.
   - **앱 진입 시 종료**: 사용자가 앱을 직접 실행하거나 포그라운드로 복귀하면 소리/진동 즉시 중단.
   - **자동 종료**: 사용자 반응이 없어도 **20초 후** 시스템이 자동으로 모든 알람을 중단.
2. **모드 결정 조건**:
   - `벨소리(Ringtone)` 설정 시: 무조건 **전체화면 모드** (인지 및 중단 동작 필요).
   - `알림음/진동` 설정 시: 유저가 선택한 모드(팝업/전체화면)를 따름.
   - **Z 플립 5 최적화**: 기기가 잠겨 있거나 접힌 상태에서도 알람 가시성을 최우선으로 보장합니다.
3. **UI 차별화**:
   - 전체화면: [타이틀 + 감성 문구] 모두 표시.
   - 팝업: [타이틀]만 강조하여 노출하여 시야 방해 최소화.
